# 🎯 核心命令（记住这个就行）
```bash
# 获取证书 + 自动配置Nginx
sudo certbot --nginx -d 你的域名
```

# 🔄 日常维护
```bash
# 查看所有证书
sudo certbot certificates

# 续期证书（自动）
sudo certbot renew

# 测试续期（不实际执行）
sudo certbot renew --dry-run
```

# ⚙️ Nginx 操作
```bash
# 修改配置后必须做的三步：
1. sudo nginx -t           # 测试配置
2. sudo systemctl reload nginx  # 平滑重载
3. curl https://你的域名   # 验证
```

# 🗑️ 删除证书
```bash
sudo certbot delete --cert-name 域名
```

# 📍 证书位置
```
/etc/letsencrypt/live/你的域名/
├── fullchain.pem  ← Nginx用这个
└── privkey.pem    ← 私钥
```
