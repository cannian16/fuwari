没有像别的数据库一样可以选择数据库，每一个数据库文件就是一个数据库(后缀一般为sqlite、db)
# CLI
## 基础操作
```bash
sqlite3 [path/to/db] # 打开数据库
.help       # 显示所有命令帮助
.open filename.db     # 打开/创建数据库文件
.databases            # 显示当前数据库
.tables     # 显示数据库里所有的表
.schema [table_name] # 显示表结构
select * from user  # 直接运行SQL语句
.exit       # 退出程序

## 查询显示设置
```bash
.headers on|off       # 显示/隐藏列标题（类似 MySQL 的 \G）
.mode <模式名>          # 设置输出模式,MODE 可以是：csv, column, html, insert, line, list, tabs, tcl
.mode column         # 类似 MySQL 的表格输出
.width NUM1 NUM2 ... # 设置列宽（配合 column 模式使用）
.nullvalue STRING    # 设置 NULL 的显示字符串
.show # 可以显示当前设置
```

```bash
#导出sql
sqlite3 app.db .dump > dump.sql
#导入sql
sqlite3 app.db <dump.sql
```
