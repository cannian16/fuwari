## RGB（光的三原色）
这个简单每个像素通过调整红绿蓝三原色混合形成所有颜色

## HSL（色相/饱和度/明度）
有些时候会叫色相叫色度
### 色相-H
- **定义**：颜色的基本属性，就是我们平常说的“这是什么颜色”，比如红色、橙色、绿色、蓝色等。
- **表示**：通常用一个 0° 到 360° 的色轮来表示。
  - `0°`：红色
  - `120°`：绿色
  - `240°`：蓝色
  - `360°`：回到红色
### 饱和度-S
- **定义**：颜色的纯度或鲜艳程度。
- **表示**：0% 到 100%。
  - `100%`：纯色，颜色最鲜艳。
  - `50%`：颜色中等鲜艳。
  - `0%`：完全没有颜色，变成不同深浅的灰色。
### 明度-L（ps里是B）
- **定义**：颜色的明亮程度。
- **表示**：0% 到 100%。
  - `100%`：最亮，无论色相和饱和度如何，都会变成白色。
  - `50%`：颜色本身的正常明度。
  - `0%`：最暗，变成黑色。

这意味着你想要白-黑的所有灰度值，只需要把饱和度设为0，色相这个情况下多少都没用，只需要调整明度即可。
你想从纯蓝-浅蓝所有的色，也是同理，色相饱和度调整好，直接调整明度

## Lab
### L通道 - 明度
- **定义**：代表颜色的**明度**或**亮度**。
- **范围**：从 **0 到 100**。
  - `L* = 0`：纯黑色
  - `L* = 100`：纯白色
- 这类似于 HSL 中的明度，但更符合人眼的视觉感知。

### a通道 - 红绿轴
- **定义**：代表颜色在**红色和绿色**之间的位置。
- **范围**：通常从 **-128 到 +127**（不同实现有差异）。
  - **`+a` 值**：表示颜色偏向**红色**（值越大越红）。
  - **`-a` 值**：表示颜色偏向**绿色**（值越小越绿）。
  - **`a = 0`**：表示中性（既无红色也无绿色倾向）。

### b通道 - 黄蓝轴
- **定义**：代表颜色在**黄色和蓝色**之间的位置。
- **范围**：通常从 **-128 到 +127**。
  - **`+b` 值**：表示颜色偏向**黄色**（值越大越黄）。
  - **`-b` 值**：表示颜色偏向**蓝色**（值越小越蓝）。
  - **`b = 0`**：表示中性（既无黄色也无蓝色倾向）。

你想获得灰度值，只需把a，b都调整到0中性，然后调整L

## CMYK
AI的

### 一、CMYK 是什么？它的原理

**CMYK** 是一种**减色模型**，意思是它通过**吸收（减去）** 白光中的某些颜色光，来呈现我们看到的颜色。它主要用于印刷领域，比如杂志、海报、宣传册等。

*   **原理**：从白色（例如一张白纸，它反射所有光）开始，当盖上油墨时，油墨会**吸收**掉白光中的某些部分，未被吸收的光被反射回我们的眼睛，我们就看到了颜色。
*   **名称来源**：CMYK 代表四种用于印刷的油墨颜色：
    *   **C - Cyan**：青色，一种蓝绿色。
    *   **M - Magenta**：洋红色，一种粉红色。
    *   **Y - Yellow**：黄色。
    *   **K - Key**：定位套版色（黑色）。

**为什么需要 K（黑色）？**
理论上，将 **C、M、Y** 三种油墨等量混合应该能产生黑色。但现实中：
1.  **成本**：用三种昂贵的彩色油墨来制造黑色非常不经济。
2.  **纯度**：混合产生的是一种浑浊、偏棕的深灰色，而不是纯正的黑。
3.  **细节与稳定性**：印刷细小文字时，用三种油墨精确套印几乎不可能，稍有偏差就会模糊。使用一种黑色油墨则清晰稳定。

因此，引入了纯黑色的 K 版来解决以上问题。“Key”在这里指的是在印刷中起关键定位作用的印版。

---

### 二、CMYK 的表示方法

CMYK 通常用四种油墨的**百分比**来表示。

*   **范围**：每种颜色的取值范围是 **0% 到 100%**。
    *   `0%` 表示完全不使用该种油墨。
    *   `100%` 表示使用该种油墨的最大浓度。

**颜色示例：**

| 颜色示例 | CMYK 值 | 解释 |
| :--- | :--- | :--- |
| 纯白色 | `cmyk(0%, 0%, 0%, 0%)` | 所有油墨都为0%，纸张本身的白色。 |
| 纯黑色 | `cmyk(0%, 0%, 0%, 100%)` | 仅使用100%的黑色油墨，最经济纯正。 |
| 纯红色 | `cmyk(0%, 100%, 100%, 0%)` | 混合100%的洋红(M)和100%的黄(Y)。（吸收绿光和蓝光，反射红光） |
| 纯绿色 | `cmyk(100%, 0%, 100%, 0%)` | 混合100%的青(C)和100%的黄(Y)。 |
| 纯蓝色 | `cmyk(100%, 100%, 0%, 0%)` | 混合100%的青(C)和100%的洋红(M)。 |
| 深灰色 | `cmyk(0%, 0%, 0%, 50%)` | 使用50%浓度的黑色油墨。 |

### 三、RGB 与 CMYK 的关系与转换

**1. 关系：互补色**
RGB 和 CMYK 在色轮上大致是互补的关系。
*   **RGB 的三原色** 是 **红、绿、蓝**。
*   **CMYK 的三原色** 是 **青、洋红、黄**，它们恰好是 RGB 的补色：
    *   青色 (C) 是 红色 (R) 的补色（吸收红光）
    *   洋红 (M) 是 绿色 (G) 的补色（吸收绿光）
    *   黄色 (Y) 是 蓝色 (B) 的补色（吸收蓝光）

**2. 转换**
在设计软件（如 Photoshop）中，可以进行 RGB 和 CMYK 的模式转换。但这是一个**有损过程**，因为两者的色域不同。

**3. 核心问题：色域差异**
*   **色域**：一个色彩模型所能表示的颜色范围。
*   **RGB 色域**（基于发光）比 **CMYK 色域**（基于反光）**更广**。这意味着屏幕上许多鲜艳、明亮的颜色（如亮蓝、亮绿）是无法用四色油墨印刷出来的。

当你在屏幕上设计一个用于印刷的作品时，如果使用了过于鲜艳的 RGB 颜色，在转换成 CMYK 后，这些颜色会变得**暗淡、发灰**，这就是所谓的“溢色”。

### 四、应用与工作流程

**“为什么我的印刷品和屏幕上看的不一样？”**
这就是因为没有正确处理 RGB 和 CMYK 的转换。

**标准的印刷设计工作流程：**
1.  **初始设定**：在开始设计前，就将文档的色彩模式设置为 **CMYK**。这在 Photoshop, Illustrator 等软件中都可以完成。
2.  **使用 CMYK 色板**：在整个设计过程中，只从 CMYK 色板库中选择颜色，确保你所见的颜色都在可印刷的范围内。
3.  **软打样**：在屏幕上模拟 CMYK 颜色的显示效果。
4.  **输出**：直接生成用于印刷的 CMYK 分色文件。

**对于普通用户：**
如果你需要打印照片，打印机内部会自动将你的 RGB 图片数据转换为 CMYK 数据来控制墨盒。但为了获得最佳效果，专业领域必须手动控制这个过程。

### 总结与对比

| 特性 | **RGB（加色法）** | **CMYK（减色法）** |
| :--- | :--- | :--- |
| **原理** | 光的叠加 | 光的吸收（减去） |
| **主要用途** | 显示器、电视、数码设备 | 印刷、打印、绘画 |
| **原色** | 红 (Red)、绿 (Green)、蓝 (Blue) | 青 (Cyan)、洋红 (Magenta)、黄 (Yellow)、黑 (Key) |
| **初始状态** | 黑（无光） | 白（白纸） |
| **最终状态** | 白（全光叠加） | 黑（理论上全油墨混合） |
| **色域** | 较广，多鲜艳色 | 较窄，颜色相对暗淡 |
| **文件格式** | 适用于屏幕显示（JPG, PNG, GIF） | 适用于印刷（PDF, EPS, TIFF） |

**简单比喻：**
*   **RGB** 就像在黑暗的房间里使用**红、绿、蓝三支手电筒**互相照射混合颜色，光叠加越多，越亮，最终变成白光。
*   **CMYK** 就像一位画家，他有一管**青色**颜料（能吸收红光）、一管**洋红**颜料（能吸收绿光）和一管**黄色**颜料（能吸收蓝光）。他在白纸上混合这些颜料，每加一种颜料，就从白光中吸收掉一种颜色，混合越多，吸收的光越多，颜色越暗，最终趋向黑色。
